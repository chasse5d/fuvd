> МБОУФГБУЗООО *"Официально не существует (это исправисимо), но уже учит"*

## 📖 О проекте
Сайт вымышленного высшего учебного заведения постепенно разрабатывается тремя людьми в юмористическом контексте

## Возможности
- **Тёмная/Светлая тема** с сохранением выбора в `localStorage`
- **Автоматический слайдер** с плавным переключением и навигацией по точкам
- **Адаптивная вёрстка** (Mobile-First подход, CSS Grid + Flexbox)
- **БЭМ-методология** для изоляции стилей и лёгкой поддержки
- **Микроанимации** при наведении на кнопки, карточки и заголовок
- **Vanilla JS** без зависимостей и фреймворков

## Тех. стек
| Компонент | Описание                                                     |
|:---------:|:------------------------------------------------------------:|
| `HTML5`   | Семантическая разметка, доступность (ARIA)                   |
| `CSS3`    | Кастомные свойства, Grid, Flexbox, `clamp()`, `aspect-ratio` |
| `JS`      | DOM-манипуляции, таймеры, события, `localStorage`            |
| `Git`     | Контроль версий, совместная работа, деплой                   |
